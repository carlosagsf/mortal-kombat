Este é o meu primeiro projeto Front-End, desenvolvido utilizando HTML5 e CSS3. A proposta do projeto foi criar uma página temática inspirada no filme Mortal Kombat 2, apresentando informações como pôster, gênero, sinopse e um botão interativo para assistir ao trailer oficial.

Durante o desenvolvimento, foram aplicados conceitos fundamentais de Front-End, incluindo:

Estruturação semântica com HTML;

Estilização moderna com CSS;

Flexbox para alinhamento de elementos;

Responsividade com Media Queries;

Efeitos de hover em botões;

Personalização visual com imagens de fundo;

Ajustes para impressão da página (@media print).

O projeto também serviu como prática para organização de código, separação entre estrutura e estilo, além de adaptação da interface para diferentes tamanhos de tela.

Tecnologias utilizadas:

HTML5
CSS3

Esse projeto representa o início da minha jornada no desenvolvimento Front-End e marca meus primeiros passos na criação de interfaces web responsivas e estilizadas.
